Detailed Updates

This is the section listing every single time the bot has been updated, big or small. The criteria over the years has changed for what truly classifies as an "update," and what the tiers of updates mean. So, I will be outlining that here.

The bot spoke its first words on July 17, 2023. Back then it was just an idea, a concept. Back then, I was still using Replit to code rather than VSCode.
Upon the initial release of the bot, I didn't have a structured update log system. The numbering was just based off vibes. Even after Update 1.2, when I added an extra tier, I didn't have a system for classification. What classified as a Tier 3 update instead of a Tier 4 update? Well, whatever felt like it took more work.

I created a structured tier system upon the release of 1.3.1. This was because 1.3.1 felt bigger than a Tier 3 update, but smaller than a Tier 4 update. So instead of adding some new thing, I redefined the tier system as follows:

Tier 1 (x.0.0.0) Updates This is the largest tier of update, an update so massive that it changes the very fabric of the game. Getting to Update 2.0 is like advancing from where humanity is now to a Type 1 Civilization.
The update must be something that completely changes the game, for example a full UI or mobile app with live statistics, but at that point I'm practically creating an indie game.
Tier 2 (0.x.0.0) Updates These are major updates. They add several features that are interdependent. Starting Update 1.2 onwards, I try not to bundle up random features, and create some sort of theme connecting the additions of these updates.
Tier 3 (0.0.x.0) Updates These are minor updates. They add one feature/concept to the game, and all the commands related to it. Examples include radars, autotax, or capitals. They change gameplay by adding more variety of strategy, but not to the extent that Tier 2 updates do.
Tier 4 (0.0.0.x) Updates These are simple patch fixes. I still announce them to notify users when some bug is fixed. This update tier is also for extremely minor additions, for example adding a small, cosmetic command.
