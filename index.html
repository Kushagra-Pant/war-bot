Welcome to War Bot!

This is a passion project I have been building since 2023. It is a command-based strategy game run by a discord bot where you invade countries, forge alliances, and develop economies. This web page contains details of everything to do with the bot. Feel free to use the navigation bar at the top right to learn more about the bot.

Update 1.5 is Here!
Diplomacy has finally arrived...

There are so many new features that were added in this update, but the major ones are alliances, treaties, and automatic population growth. The treaty system in particular has been completely revamped and upgraded! Full details on this update can be found in Detailed Updates. Some features promised in this update have been delayed to future updates, including partial treaties, military modernization, and the trusted diplomacy setting.
