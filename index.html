Welcome to War Bot!

This is a passion project I have been building since 2023. It is a command-based strategy game run by a discord bot where you invade countries, forge alliances, and develop economies. This web page contains details of everything to do with the bot. Feel free to use the navigation bar at the top right to learn more about the bot.

Newest Update
Update 1.5.2 - Command & Control

This update adds two new features to make handling large militaries significantly easier: Theaters, which allow you to group provinces together and focus on a specific front, and military groupings, which allow you to group multiple unit types together and attack/move the group as a whole.
In the future, commands may be added to make making or handling these easier, and new unit types may be added to enhance gameplay!
