Welcome to War Bot!

This is a passion project I have been building since 2023. It is a command-based game run by a discord bot where you invade countries, forge alliances, and develop economies. This web page contains a nice summary of everything to do with the bot. Feel free to use the navigation bar at the top right to learn more about the bot.

Latest Update - Revolutions!
Update 1.4.3

The latest update adds Revolutions - you can change your ideology to any other in the game for a price of 1/3 of your stability! This is a useful feature in case you'd like to change to a distant ideologies but don't have the political power to do so. At high stabilities, revolution will be damaging. To cause a revolution, do -revolt <ideology>
Other changes were made, like a small reworking of naval combat, inclusion of current balance in -top power, a change to the stability gain formula of statues, and a bombardment buff.
You can find other additions to this update in Detailed Updates.
